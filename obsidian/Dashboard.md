### 1. Overview
The Dashboard contains the main features of the app, and provides navigation options to the user. It has two views: the tasks view, and the calendar view.

**Task view:** 
- shows all tasks in a list style

**Calendar view:**
- shows the same tasks, but in a calendar like view
- has 3 variant - day, week, month
